FlaskLib

Test project

This application allows users to manage information about books and writers. List of all books can be found on the Bookshelf tab. Information about authors is available on click. List of all writers and their books can be found on the Writers tab.

Features

  • Design was created using Bootstrap and Visual Studio web app template
  • Only logged in users can edit and delete items
  • Information about book authors is loading using the standart AJAX functions from jQuery
  • Security: password hashes are stored in database instead of passwords
  • Cool loading animation makes design a bit more responsive
  • Easy books adding: one input field for writers, suggestions
  • Unexistent writers are automatically created from input field in book edit menu
  • User data is validated using WTForms validators
  • Case-insensitive search, results from books and authors

Things used

  • Flask
  • Flask-login
  • SQLAlchemy
  • WTForms
  • SQlite database inside the application
  • jQuery
  • Bootstrap
